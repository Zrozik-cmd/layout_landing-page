Description

Responsive landing page for the MyBike brand, created from a Figma design.
The project focuses on clean layout, semantic HTML structure, and modern responsive design.

Technologies
HTML5 – semantic structure
SCSS (SASS) – modular styles with variables and mixins
BEM methodology – for clean and maintainable class naming
CSS Grid & Flexbox – for adaptive and responsive layout
Responsive Design – optimized for desktop, tablet, and mobile

Features
Fully responsive layout (desktop / tablet / mobile)
Mobile navigation menu with smooth animation
Well-structured SCSS and BEM architecture
Implemented sections: Hero, Compare Bikes, Details, Contacts
