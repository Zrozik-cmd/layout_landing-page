# Modern Landing Page — Responsive Layout

![Landing Page Preview](preview.mov)

A clean, modern, and fully responsive landing page built from a Figma/PSD design (or as a practice project). Built with pure HTML, CSS/SCSS.

## Features

- Fully responsive layout (320px → 1920px+)
- Semantic HTML5 markup
- Modern CSS3 / SCSS with Flexbox & Grid
- Smooth scrolling & scroll animations (AOS or custom CSS)
- Optimized images and fast loading
- Cross-browser compatible (Chrome, Firefox, Safari, Edge)
- Ready for CMS integration or further development
